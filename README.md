# 📚 AI Study Assistant

AI Study Assistant is a Flask-based web application that allows users to upload PDF notes and get summarized content instantly.

## Features

- Upload PDF files
- Extract text from PDF
- Generate summary
- Clean dark UI
- Flask backend

## Technologies Used

- Python
- Flask
- HTML
- CSS
- pdfplumber

## How to Run

1. Install requirements

pip install -r requirements.txt

2. Run app

python app.py

3. Open browser

http://127.0.0.1:5000
